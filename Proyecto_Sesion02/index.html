Escribe la diferencia entre px y rem como unidades de medida. Incluir dos fuentes de Google fonts en tu proyecto.

Diferencia entre px y rem:
px (píxeles): Es una unidad de medida que representa un píxel en la pantalla. Es una unidad estática y no cambia en relación con otros elementos en la página.
Por ejemplo, si defines el tamaño de fuente de un elemento como font-size: 16px, será 16 píxeles, independientemente de otros estilos o tamaños de fuente en la página.
rem (root em): Es una unidad de medida relativa con respecto al tamaño de fuente del elemento raíz (normalmente el tamaño de fuente del elemento ).
Por ejemplo, si defines el tamaño de fuente de un elemento como font-size: 1.5rem, será 1.5 veces el tamaño de fuente del elemento raíz. Si el tamaño de fuente del
elemento raíz es 16 píxeles, entonces el tamaño de fuente del elemento será 24 píxeles (16 * 1.5).
En resumen, px es una unidad estática y rem es una unidad relativa que toma como referencia el tamaño de fuente del elemento raíz.
